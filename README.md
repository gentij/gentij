## 🚀 Featured Projects

### ⚙️ Lunie

A self-hosted workflow automation engine — think *Zapier, but open and yours*.

* 🧠 CLI + API architecture
* 🐳 Docker-based deployment (server + workers)
* 🔐 API token authentication system
* 🔄 Workflow engine with triggers, steps, and runs
* 🧩 Built with Go, NestJS, Prisma, PostgreSQL, Redis

> Designed for developers who want full control over their automation stack.

---

### 🎵 Stemmer

A desktop app for **AI-powered audio stem separation** — fully local, no cloud required.

* ⚡ Built with Rust + Tauri
* 🧠 Uses ONNX models for inference
* 🎧 Splits audio into vocals, drums, bass, and more
* 🖥 Cross-platform desktop app (macOS, Windows, Linux)

> Focused on performance, privacy, and simplicity.

---

### 🧪 stem-splitter-core

The core engine behind Stemmer — a pure Rust library for audio processing.

* 🦀 Written in Rust for performance
* 🤖 ONNX Runtime integration for model inference
* 🎼 Handles audio decoding, chunking, and reconstruction
* 📦 Designed as a reusable SDK + CLI

> Built to make local AI audio processing fast and accessible.
